# Configuration

- **`config.yaml`** — main gateway and connector settings. Start from the version in this repo and replace `YOUR_*` placeholders.
- **Examples:** `config_example.yaml`, `dialogflow_cx_example.yaml`, `aws_lex_example.yaml`.

Full setup (secrets, Google auth, Webex, TLS) is documented in the repository root **[README.md](../README.md)**.
